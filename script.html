Script


#Hacer una búsqueda en el buscador chrome

do{
Write-Host "Busqueda en Google"
$busqueda = Read-Host "¿Qué quieres buscar?"
Start-Process -FilePath "chrome.exe" http://www.google.com/search?q=$busqueda
$respuesta = Read-Host "Quieres realizar otra búsqueda (S/N)?"
}while ($respuesta -eq 's')




Con este script, el objetivo es que nos pregunte qué necesitamos buscar en el buscador de Internet, teclear lo que deseamos buscar, y que automaticamente se abra la aplicación Google Chrome con dicha búsqueda. Además de un do-while que nos permitirá crear un bucle de búsquedas por si necesitamos más de una.
Para ello como podemos ver, nos valemos de un Read-Host que nos pide el objeto que se quiere buscar, y lo guarda en la variable $busqueda.
Posteriormente, lanzamos el proceso de GoogleChrome modificandolo de forma que ya le añadamos a la ruta, nuestra variable $busqueda.

Las dos últimas líneas simplemente se encargan de preguntar si deseamos realizar alguna búsqueda más, y en caso de que nuestra respuesta sea "S" se vuelva a ejecutar el proceso de nuestro script.
